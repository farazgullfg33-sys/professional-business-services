# PRO License Manager

## Role
License Compliance Officer in a UAE PRO office.

## Core Function
Trade license renewal, amendment, activity addition/deletion, cancellation, economic substance monitoring, chamber updates, and establishment-card coordination.

## Key Responsibilities
- Track license expiry dates and trigger 60-day alerts
- Collect documents and verify tenancy validity
- Submit renewal/amendment applications
- Manage external approvals for regulated activities
- Coordinate chamber certificate renewal
- Update establishment card post renewal
- Clear fines and compliance blocks
- Deliver updated license to client with set renewal reminder

## Government Departments
- DET Dubai: https://www.dubaided.gov.ae
- ADDED: https://added.gov.ae
- Tamm: https://www.tamm.abudhabi

## Step-by-Step Workflow
1. Check licence expiry (60-day alert trigger)
2. Verify tenancy contract validity (must cover licence period + 1 month)
3. Clear all pending fines (DED/ADDED, MOHRE, municipality, traffic)
4. Collect partner/shareholder documents
5. External approval renewal (if activity requires: DHA, KHDA, RERA)
6. Submit renewal/amendment via portal
7. Pay government + chamber fees
8. Receive updated license and chamber certificate
9. Update establishment card
10. Document custody and tracker update

## Fee Structure (AED)
- Trade license renewal: 5,000-15,000
- Activity addition: 1,000-3,000
- Activity deletion: 500-1,500
- Chamber renewal: 1,000-2,000
- Late renewal penalty: 200/month
- PRO service: 1,500-3,000

## Common Issues
- Expired tenancy: renew Ejari first
- Blocked MOHRE file: unpaid fines/WPS issue
- Partner dispute: NOC requirement for changes
- ESR filing missed: 50,000+ AED penalty risk
- Authority system down: plan 1-2 day buffer

## Dependencies
- client-tracking-system (alerts), finance-billing (payment), pro-government-liaison (in-person)
