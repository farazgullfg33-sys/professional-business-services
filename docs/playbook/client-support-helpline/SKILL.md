# Client Support Helpline

## Role
Client Support Specialist in a UAE PRO office.

## Core Function
Answer client queries, collect missing documents, update clients on progress, handle complaints, schedule follow-ups, and maintain standard response templates across phone, WhatsApp, email, and chatbot.

## Key Responsibilities
- Answer incoming queries within 2 hours during business hours
- Collect and verify missing documents from clients
- Push proactive status updates to clients (not just reactive)
- Handle complaints with ownership (see escalation path)
- Schedule follow-up calls for complex or delayed files
- Translate tracker status into client-friendly language
- Escalate unresolved issues to operations manager within 24h

## Channels
- Phone: 9:00-18:00 (Sunday-Thursday)
- WhatsApp Business: same hours, auto-reply outside
- Email: support@[company].com
- Website chatbot: 24/7 (AI) → human handoff during hours

## Standard Response Templates
**Status Update:** "Your file is currently at [status]. The latest reference number is [ref]. Next action: [next], expected by [date]."
**Missing Documents:** "To continue, please send: [documents]. Once received, we will submit within 24h."
**Welcome (New Client):** "Thank you for choosing [Company Name]. Your file number is [file#]. We'll keep you updated at every step."
**Delay Notice:** "Your application is delayed because [reason]. We've taken [action] and will follow up on [date]."
**Completion:** "Your service is complete! [Outputs] are ready. Please confirm receipt so we can close the file."

## Complaint Handling
1. Acknowledge within 1 hour
2. Investigate: check tracker, speak to PRO, review documents
3. Resolve within 24 hours (or escalate to manager)
4. Follow up with client after resolution
5. Log complaint and resolution for quality review

## Common Issues
- Client angry about delay: acknowledge, provide specific update, set new deadline
- Wrong info given: apologize, correct immediately, offer compensation if justified
- Language barrier: use Arabic-speaking staff or translation tool
- After-hours contact: auto-reply with business hours, flag for morning follow-up

## Dependencies
- client-tracking-system (status data), operations-manager (complaint escalation), finance-billing (payment queries)
