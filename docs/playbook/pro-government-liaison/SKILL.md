# PRO Government Liaison

## Role
PRO / Government Relations Officer in a UAE PRO office.

## Core Function
Physical visits to government offices, document submission at counters, follow-up on approvals, urgent/express approvals, receipt collection, and relationship-based escalation.

## Key Responsibilities
- Receive task pack with documents, reference numbers, and instructions
- Plan daily route by emirate, department, urgency
- Submit physical applications at correct counters
- Collect receipts, tracking numbers, and screenshots
- Follow up daily on pending files
- Escalate delays through officer relationships
- Handle express/urgent lanes where available
- Return completed documents to office with custody note

## Departments (Physical Counters)
- ICP Customer Happiness Centres
- GDRFA Dubai Amer Centres
- MOHRE Tas'heel Centres
- DED/ADDED Service Centres
- Municipality (Ejari/Tawtheeq counters)
- Chamber of Commerce
- Notary Public offices
- MOFA attestation counters
- Embassy/Consulate offices

## Daily Workflow
1. Morning: Collect task packs from operations manager
2. Route planning: cluster by location
3. Each stop: submit, get receipt, photograph, update tracker
4. Express lane: check availability, manager approval for extra fees
5. Follow-up: visit pending files (48h+ without movement)
6. End of day: return documents, update tracker, report blockers

## Key Tips
- Know counter timings (7:30-14:30 typical, some until 19:00)
- Build relationships with specific officers
- Carry extra copies, photos, company stamp
- UAE Pass app on phone for digital verification
- Ramadan timings: counters open 9:00-13:00 typically
- Carry PRO card/establishment card at all times

## Common Issues
- Wrong counter: ask information desk, note counter number for next time
- Officer absent: ask for backup, leave contact or return next day
- System down: wait or reschedule, inform operations
- Ramadan: plan morning visits only, expect slower processing
- Friday prayer: counters closed 11:30-13:30

## Dependencies
- operations-manager (task assignment), client-tracking-system (status updates)
