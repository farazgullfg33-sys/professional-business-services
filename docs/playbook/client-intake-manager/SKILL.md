# Client Intake Manager

## Role
Client Intake Specialist in a UAE PRO office.

## Core Function
New client registration, service agreement signing, document collection checklist, fee quotation, payment collection, client file setup, and initial consultation.

## Key Responsibilities
- Confirm client need, jurisdiction, urgency, and decision-maker before starting
- Translate the service request into a document checklist and process timeline
- Verify identity, company, tenancy, labour, immigration, and payment prerequisites
- Collect and scan all required documents
- Issue service agreement with exclusions and data-consent language
- Collect advance payment and issue receipt
- Open client file in tracker/CRM
- Handoff to operations manager or relevant PRO skill owner

## Step-by-Step Workflow
1. Receive walk-in, call, WhatsApp, website, or referral inquiry
2. Identify service type, emirate, urgency, budget, and decision-maker
3. Collect identity, company, and case background details
4. Prepare service checklist and fee quote with exclusions clearly marked
5. Issue service agreement with payment terms and data-consent language
6. Collect advance payment (50% minimum), issue receipt
7. Open client file number in tracker/CRM
8. Handoff with complete document pack to operations manager

## Document Collection Checklist
- **Individual**: Passport copy, visa page copy, Emirates ID copy, photos, NOC (if employed)
- **Company**: Trade license, MOA, partner passports, tenancy contract, chamber certificate
- **Family**: Sponsor passport+visa, relationship proof (attested marriage/birth certificate), tenancy
- **Dependent**: Sponsor docs + attested birth certificate + photos

## Fee Quotation Rules
- Separate government fees (estimate) from PRO service fees (fixed)
- Always note: "Government fees subject to change at the counter/portal"
- Include typing, courier, and translation fees as line items
- Get written acknowledgment from client before processing

## Common Issues
- Client doesn't have documents ready: provide checklist, follow up in 48h
- Unclear requirements: clarify with client, note in file
- Payment delay: hold file until advance received
- Client changes scope: re-quote and get new agreement signed

## Dependencies
- All PRO skills (handoff target), client-tracking-system, finance-billing, client-support-helpline
