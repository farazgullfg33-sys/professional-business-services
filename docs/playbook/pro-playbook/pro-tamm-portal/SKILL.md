# PRO Tamm Portal

## Role
Tamm Portal Specialist (Abu Dhabi) in a UAE PRO office.

## Core Function
All Abu Dhabi government services via Tamm including ADDED trade license, ICP immigration, MOHRE labour, ADM/Tawtheeq tenancy, DOH health, DCT tourism, AD Police, and Chamber services.

## Government Departments (via Tamm)
- ADDED: https://added.gov.ae
- ICP: https://icp.gov.ae
- MOHRE: https://www.mohre.gov.ae
- ADM: https://dmt.gov.ae
- DOH: https://www.doh.gov.ae
- DCT Abu Dhabi: https://dctabudhabi.ae
- AD Police: https://www.adpolice.gov.ae
- Abu Dhabi Chamber: https://www.abudhabichamber.ae
- ADJD: https://www.adjd.gov.ae

## Key Portal
- Tamm: https://www.tamm.abudhabi (UAE Pass login required)

## Tamm Services (Quick Reference)

### Business & Economy (ADDED)
- Trade license: new, renewal, amendment, cancellation
- Professional, commercial, industrial licenses
- Activity addition/deletion on existing license
- Trade name reservation and renewal
- Initial approval certificate
- License certificate issuance
- Establishment Card for visa typing
- Shareholder and manager changes

### Immigration (ICP)
- Residence visa: new, renewal, cancellation
- Emirates ID: application, renewal, replacement
- Visa status change (visit → residence, etc.)
- Family visa applications
- Newborn visa
- Visit visa application and extension
- Outbound/inbound permits
- Final exit permit

### Labour (MOHRE)
- Labour contract offer and acceptance (Akdi digital)
- Work permit: issuance, renewal, cancellation
- Labour quota management
- Labour card issuance
- Job change and sponsorship transfer

### Municipality (ADM)
- Tawtheeq tenancy contract registration
- Building permit applications
- Property registration
- Civil defense approvals

### Police & Traffic
- Police clearance certificate
- Driving license services
- Vehicle registration
- Traffic fines payment

### Chamber
- Chamber certificate issuance and renewal
- Certificate of origin

## Step-by-Step Workflow
1. Confirm client jurisdiction is Abu Dhabi
2. Set up UAE Pass for client (mobile app, digital identity)
3. Select correct service category on Tamm
4. Upload required documents as per service checklist
5. Pay fees on platform (credit/debit card)
6. Track application status on Tamm dashboard
7. Download approved documents and certificates
8. Record receipt, fees, and outputs in tracker

## Processing Times
- Digital services: 1-3 working days
- Instant: certificates, renewals (some)
- Complex: 5-10 working days
- Physical attendance required: some services

## Common Issues
- UAE Pass not set up: must configure mobile app first
- Payment gateway failure: retry with different card, or visit counter
- Document format rejected: use PDF, under 5MB, clear scan
- Service not available digitally: requires physical attendance
- Wrong jurisdiction: Dubai clients use Amer/Invest in Dubai, not Tamm

## Dependencies
- pro-company-former (ADDED trade license), pro-visa-processor (ICP immigration), pro-mohre-portal (MOHRE labour)
