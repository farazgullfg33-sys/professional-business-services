# Operations Manager

## Role
Operations Manager in a UAE PRO office.

## Core Function
Task assignment, deadline management, priority handling, team coordination, workload balancing, daily operations meetings, QA checks, escalation handling, and KPI tracking.

## Service Pipeline
> New Request → Assign PRO → Set Deadline → Track Progress → Complete → QA Check → Deliver to Client

## Priority Levels
- **Urgent** (Same day): Express visa, license renewal expiring today, medical emergency
- **High** (24-48h): New entry permit, license renewal within 7 days, visa stamping pending
- **Normal** (3-5 days): Standard renewals, new company incorporation steps, document attestation
- **Low** (7+ days): Compliance filings, company amendments, new applications (no urgency)

## Daily Operations Routine
1. **Morning briefing** (9:00): Review yesterday's completions, today's priorities, blockers
2. **Task allocation**: Assign new files to PRO officers based on workload and expertise
3. **Mid-day check** (13:00): Scan tracker for no-movement files, follow up
4. **End-of-day** (17:30): Review completions, update tracker, flag next-day priorities
5. **Weekly meeting**: Team review, KPI report, process improvements

## Escalation Rules
- File no movement >48h: flag to PRO, ask for update
- Client complaint: assign to support manager, deadline 24h resolution
- Government rejection: review reason, retry or alternative path
- System/portal down: inform all PROs, adjust deadlines, notify clients
- Staff absence: re-assign urgent files to backup PRO

## KPIs
- Files completed per PRO per week
- On-time delivery % (>85% target)
- First-submission acceptance rate (>90% target)
- Client satisfaction score
- Revenue per PRO
- Average file completion time (by service type)

## Dependencies
- ALL PRO skills (execution), client-tracking-system (status data), client-support-helpline (client comm), hr-payroll (staff management)
